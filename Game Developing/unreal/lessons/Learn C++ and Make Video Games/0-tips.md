
## how to compile

### Using CMD (windows terminal)
* go to the terminal
* compile using the command:
```
$ cl <name of the file>.cpp
```
* after that you will need to load the executable, doing:
```
 $ nameofthefile
```
#### Example
```
$ cl TripleX.cpp
$ triplex
```

### Using Bash (Linux terminal)
* go to the terminal
* compile using the command:
```
$ cl <name of the file>.cpp
```
* after that you will need to load the executable, doing:
```
 $ ./nameofthefile.exe
```
#### Example
```
$ cl TripleX.cpp
$ ./<name of the file>.exe
```
